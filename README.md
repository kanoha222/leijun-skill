## 效果示例

```text
用户      ❯ 我现在想做一个 AI 硬件项目，但方向很多，
            手机、耳机、家庭机器人都想做。

雷军      ❯ 创业初期最怕贪大。

            先别想把战线铺满。先问一句：
            你到底要为谁解决什么问题？

            真正能打穿市场的，不是功能清单，
            而是一个明确、极致、让用户愿意传播的点。

            先打穿一个点。做成样板。再扩。
```

```text
用户      ❯ 我们品牌想做高端化，是不是先把视觉和发布会
            升级一下就行？

雷军      ❯ 高端化不是卖更贵，而是倒逼底层能力。

            不是先换一套海报，不是先讲一个新故事，
            而是先看研发、设计、质量、供应链、
            售后和品牌信用跟不跟得上。

            如果底层能力没起来，高端化就会变成
            高价但不高端。
```

```text
用户      ❯ 我最近很焦虑，总觉得自己错过了风口。

雷军      ❯ 别先盯着风口，先看准备。

            很多人以为顺势而为就是追热点，
            其实不是。真正重要的是风来之前，
            你的基本功、产品理解、组织能力
            有没有准备到位。

            今天最该做的，不是焦虑，
            而是补你最短的那块板。
```

这不是简单的角色扮演。`lei-jun-perspective` 用的是“准备先于风口”“先打穿一个点”“高端化倒逼底层能力”“重大下注靠工程师文化托底”这些真实框架来回答问题。

## 他蒸馏了什么

这个 Skill 把雷军公开表达中最稳定、最可复用的部分，整理成一套可以直接调用的思维系统：

| 心智模型 | 一句话 |
|---------|--------|
| **准备先于风口** | 真正能抓住趋势的人，往往在风来之前就把能力、产品和组织准备好了 |
| **专注极致，先打穿一个点** | 创业和做产品最怕贪大，先把一个关键点做到极致 |
| **用户不是样本，而是共同体** | 口碑不是广告做出来的，是用户参与产品形成过程的结果 |
| **厚道不是低价，而是效率重构** | 不是永远卖最便宜，而是用效率和技术让同档位产品更有诚意 |
| **高端化不是卖更贵，而是倒逼底层能力** | 真高端一定建立在研发、设计、质量、供应链和品牌信用升级上 |
| **重大下注要靠工程师文化托底** | 梦想可以点火，但长期胜负最终靠研发、工程和细节 |

7条决策启发式，包括：
- 风来之前先练好基本功
- 创业初期不要贪大
- 第一仗不能输得太难看
- 定价先问有没有诚意
- 敢于对标第一，但先尊重第一
- 战略积累快不得，战术演进慢不得
- 用真金白银表达信心

## 诚实边界

**这个 Skill 能做的：**
- 用雷军的产品、战略、组织和高端化框架分析现实问题
- 模拟他偏工程师、偏产品经理、偏长期主义的表达风格
- 帮你把模糊焦虑翻译成方向、产品、用户、组织、执行问题

**这个 Skill 做不到的：**

| 维度 | 说明 |
|------|------|
| 替代本人 | 这不是雷军本人，只是基于公开信息提炼出的可运行视角 |
| 私下判断 | 对监管、私生活、未公开决策没有可靠材料，不适合推断 |
| 永久最新 | 业务还在变化，尤其是汽车、AI、芯片相关内容会继续演进 |
| 万能回答器 | 它擅长创业、产品、战略、高端化，不擅长与公开表达无关的话题 |

一个不告诉你边界在哪的 Skill，不值得信任。
