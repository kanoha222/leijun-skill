# 雷军时间线

> 调研时间：2026-04-07
> 说明：本文件整理雷军从早年经历到最新动态的关键节点，并标注这些节点对其思维方式的影响。

## 一、关键时间线

|------|------|------------------|

| 2004 | 卓越网卖给亚马逊 | 接受阶段性退出与资本合作，理解效率与规模的价值 |
| 2007 | 金山上市后离开一线管理 | 开始更广泛地做投资和产业观察 |

| 2011 | 发布小米手机 1；同年重回金山担任董事长 | 一边证明小米方法论，一边体现其强烈的情义与责任感 |
| 2013-2014 | 小米快速成为中国头部手机品牌，生态链模式逐步展开 | “专注、极致、口碑、快”从产品方法论变成公司级方法论 |
| 2018-07-09 | 小米在港交所上市 | “价格厚道”第一次被带入资本市场，也经历了破发挫折 |
| 2019 | 公开系统阐述“我的梦想和使命” | 从企业家走向产业叙事者，开始更明确地谈制造业、科技和国家产业升级 |
| 2020-08 | 十周年演讲《一往无前》 | 开始大规模公开复盘低谷与反思，个人叙事成熟 |

| 2022 | 《小米创业思考》相关内容进一步外化 | 将内部方法论整理为更通用的创业规则库 |
| 2023-08-14 | 年度演讲《成长》 | 把个人成长、终身学习、高端化和 AI 同时拉到台前 |

| 2024-07-19 | 第五次年度演讲《勇气》 | 公开谈造车过程中的反思与勇气，个人与公司命运叙事进一步绑定 |
| 2025-03-05 | 在全国两会“代表通道”发言，强调继续走科技创新道路、高端化道路 | 从消费电子企业家进一步转向科技制造业代表人物 |
| 2025-09-25 | 第六次年度演讲《改变》 | 将高端化、造芯、造车与硬科技转型串联为一个系统性叙事 |

| 2026-03 | 新一代 Xiaomi SU7 系列发布，3 天锁单超 30,000 台 | 证明其“创始人流量 + 产品力 + 组织兑现”模式仍在发挥作用 |

## 二、阶段划分

### 1. 梦想点火期：1987-1991

- 关键词：硅谷之火、自学、编程、校园创业
- 这个阶段形成了他最根本的三件事：
  - 对“伟大公司”的向往
  - 强烈的效率意识
  - 通过自学和请教快速补齐短板的习惯

### 2. 软件工业训练期：1992-2007

- 关键词：金山、管理、软件产品、卓越网
- 这个阶段让他从热血程序员变成真正的经营者。

### 3. 小米方法论爆发期：2010-2018

- 关键词：互联网方法做制造、MIUI 社区、生态链、上市
- “专注、极致、口碑、快”与“感动人心、价格厚道”完成定型。

### 4. 反思与重下注期：2019-2023

- 关键词：梦想和使命、高端化、反思、造车、成长、AI
- 雷军从“性价比创业者”转向“长期重投入型创始人”。

### 5. 硬科技兑现期：2024-2026

- 关键词：SU7、勇气、改变、芯片、AI、人车家全生态
- 汽车、高端化和 AI 开始进入可以用经营指标检验的阶段。

## 三、最新动态（截至 2026-04-07）

### 2025-03-05

- 雷军在全国两会“代表通道”公开表示：继续坚持走科技创新道路、高端化道路。

### 2025-09-25

- 第六次年度演讲主题为《改变》，外界普遍将其视为从“互联网公司”向“硬核科技公司”转向的公开总结。

### 2026-03-24

- 小米 IR 公布 2025 全年业绩：
  - EV 交付 411,082 辆
  - 智能电动汽车等新业务经营利润转正
  - “人车家全生态”继续被放在核心战略位置

### 2026-03

- 新一代 Xiaomi SU7 系列上市，官方材料披露 3 天锁单超 30,000 台。

## 四、时间线对心智模型的支持

从时间线上可以清楚看到几条长期稳定的线索：

1. 梦想从大学阶段开始就存在，并非后期包装。
2. 工程师方法与自学能力从青年时期就已定型。
3. 小米不是一次单点创业，而是其对“互联网改造制造业”长期实验的主舞台。
4. 2019 以后，雷军明显从“做爆款产品”转向“补底层能力、做硬科技公司”。
5. 造车不是偶发跨界，而是高端化、AI、芯片与生态延伸的自然结果。
